{# 变量的使用 #}

name: {{name}}

{# 列表的使用 #}

my_list:{{my_list}}

my_list 第一个元素：{{my_list[0]}},第二个元素：{{my_list.1}}

{# 字典的使用 #}

my_dict：{{my_dict}}

姓名：{{my_dict.name}}、年龄{{my_dict["age"]}}

{# 控制代码块 #}

for循环使用: {% for item in my_list %} {{item}} {% endfor %}

使用 if 删选列表数据大于3的： {% for item in my_list %} {% if item > 3 %} {{item}} {% endif %} {% endfor %}

过滤器的使用：变量名 | 过滤器

字符串转大写：{{ name | upper }}

字符串反正：{{ name | reverse }}

列表第一个：{{ my_list | first}}、最后一个：{{ my_list | last}}
